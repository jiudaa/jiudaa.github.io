python开发环境配置

2020-12-15 | 编程

说明

Pycharm是一个很好的python集成开发工具, 但是有时候写一些简单代码时用起来并不轻便. 这里记录如何通过编辑器(notepad++和vim) 方便运行代码.

notepad++

这里我用的windows系统, 选择”运行” -> “运行”后, 在运行程序名内输入

cmd /k表示打开cmd并且运行/k后面的程序, 后面切换环境并输入自己想要的代码.
$(FULL_CURRENT_PATH) ：当前文件的路径, 为Notepad++的宏定义.
PAUSE暂停, EXIT退出.
可以“保存”该程序名和快捷键, 之后直接在编写完代码之后执行. 也可以在“运行”-> “管理快捷键”下修改.

vim

  • 环境: ubuntu 18.04
  • 在~/.vimrc用户配置文件下加入以下代码(可以通过which vim找到安装目录, 系统配置文件一般在/usr/share/vim/.vimrc), 将F5作为运行命令.
  • 一些常用配置
    语法高亮: 显示行号: 凸显当前行: 显示括号匹配: 缩进: 参考

Sublime text

  1. 设置Sublime Text的语法为python： View -> syntax ->python
  2. 设置编译环境：Tools -> Build System -> New Build System

    在打开的文件里面贴上以下代码:
    {
    “cmd”: [“/Users/mac/opt/anaconda3/bin/python”, “-u”, “$file”],
    “file_regex”: “^[ ]File "(…?)", line ([0-9]*)”,
    “selector”: “source.python”
    }
    “cmd”需要填写python程序路径, 可以用which python

    填写完成后记得保存，不要更改后缀

    完成后能在Tools ->Build System下看到自己的编译工具.

    最后打开一个python文件进行测试，直接command + B

另外也可以使用Anaconda插件 (ctrl+shift+p调用出窗口：Install Package安装)，再修改
配置Sublime Text-> preferences -> package Settings-> Anaconda -> Settings-Default.
