  6. 文件操作

  • 取文件名字(不含后缀) name = f.split('/')[-1][:-4]其中-4根据后缀自定义
  • Path(path).parent == Path(path).parents[0] 上一层文件夹 .parents[1]上两层文件夹
  • Path(path).name获取文件名(含后缀), Path(path).suffix获取后缀
  • str.strip(s)删除首位指定(多个)字符

  1. 有argparse.ArgumentParser()的代码会出错的问题
    add_argment里面的参数有分为可选和必选, ‘f’, ‘-f’,’–f’第一种是必选, 即使加了default仍染会报错. 可将’f’改为’–f’同时加上default即可

  2. 各种文件操作 yaml(load后返回的是dict)

  3. logging使用
    级别排序要知道: critical>error>warning>info>debug
    使用之前设置日志级别(小于该级别的不会显示) logging.basicConfig(level=logging.info)
    设置日志格式
    logging.Formatter('\n%(asctime)s\t%(message)s', datefmt='%m/%d/%Y %H:%M:%S')
    StreamHandler能够将日志信息输出到sys.stdout, sys.stderr 或者类文件对象（更确切点，就是能够支持write()和flush()方法的对象）。FileHandler将日志信息输出到磁盘文件上。

  4. 图像操作
    cv2.imread(file,1) 读入3通道BGR图片 (1024,1024,3)
    cv2.imread(file,0)读入单通道灰度图. (1024,1024)
    PIL.Image.open(file).convert(‘1’) 二值图像(255,0)
    PIL.Image.open(file).convert(‘L’) 灰度图像0255
    PIL.Image.open(file).convert(‘RGB’) 图像0    255
    numpy Image->tensor: torch.from_numpy(img.transpose((2, 0, 1))).float().div(255).unsqueeze(0)
    tensor -> numpy: img = tensor.mul(255).byte().cpu().numpy().squeeze(0).transpose((1, 2, 0))

    data=pd.DataFrame(a)#这时候是以行为标准写入的
    data=data.T#转置之后以列写入
  • a=[1,2,3,4]#列表a
    b=[5,6,7,8]#列表b
    c={“a” : a,
    “b” : b}#将列表a，b转换成字典

  1. json and pickle 四个功能 dumps、dump、loads、load
    json只能序列化常用数据类型列表、字典、字符串、数字. pickle可序列化任何包括类函数
    dumps把数据类型变成字符串,dump把数据类型转换成字符串并储存,loads把字符串转换成数据类型, load把文件打开从字符串转化为数据类型.
